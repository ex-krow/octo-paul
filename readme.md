# octo-paul

### a football prediction engine named after famed paul the octopus.

what does this paul predict?
paul predicts 7 spot betting markets of fixtures in europe's top ten leagues.

1. 1-X-2
2. 1X-12-X2
3. highest scoring half
4. over/under n.5
5. gg/ng
6. 1-X-2 & over n.5
7. to win to nil
